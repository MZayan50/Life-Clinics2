ربط Firebase للمزامنة السحابية
لتزامن البيانات بين الكمبيوتر والموبايل تلقائياً، اربط النظام بـ Firebase مجاناً.
1
اذهب إلى console.firebase.google.com وسجل دخول بـ Google
2
اضغط Add project → اكتب الاسم → Create
3
اضغط أيقونة Web </> → Register → انسخ firebaseConfig
4
اذهب لـ Firestore Database → Create database → Test mode
5
افتح الإعدادات ⚙️ في البرنامج → تبويب Firebase → الصق البيانات
