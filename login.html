🔥 إعداد Firebase
الصق بيانات Firebase لتفعيل المزامنة بين PC والموبايل وكل الأجهزة
1. اذهب لـ console.firebase.google.com
2. Add project → اكتب الاسم → Create
3. Web </> → Register → انسخ firebaseConfig
4. Firestore Database → Create → Test mode
5. Authentication → Email/Password → Enable
